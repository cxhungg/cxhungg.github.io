LC-3 Virtual Machine

06/28/2025

This project implements an LC-3 Virtual Machine system, including a C-based emulator, a Python interactive debugger, and an assembler. I learned how to do this through this blog.

Project Components

  • LC-3 Virtual Machine (C): A full emulator for the LC-3 instruction set, supporting memory-mapped I/O, keyboard input, and all standard instructions and traps.
  • Interactive Debugger (Python): A Tkinter-based GUI debugger for step-by-step execution, breakpoints, register/memory inspection, and console I/O simulation.
  • Assembler (Python): Converts LC-3 assembly code into object files compatible with the VM, supporting labels, directives, and all standard instructions.

Summary

  • Emulates the full LC-3 instruction set and memory model in C, including all trap routines and I/O.
  • Provides a graphical debugger for interactive program development and debugging.
  • Includes an assembler for converting LC-3 assembly to binary object files.
  • Supports breakpoints, step execution, register/memory viewing, and console I/O.

Used: C, Python, Assembly, Windows
